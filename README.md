# API labecommerce-backend

Essa é uma API desenvolvida para suportar um sistema de e-commerce. A API é capaz de gerenciar produtos, categorias, usuários, pedidos e outras funcionalidades essenciais para um sistema de e-commerce.

## Tecnologias Utilizadas

* NodeJS

* Typescript

* Express

* Postman

## Endpoints
* GET /api/users: retorna todos os usuários

* POST /api/users: cria um novo usuário

* POST /api/products: cria um novo produto

* GET /api/products: retorna todos os produtos (funcionalidade 1)

* GET /api/products: retorna todos os produtos (funcionalidade 2)

* PUT /api/products/:id: atualiza um produto pelo ID

* POST /api/purchases: cria uma nova compra

* DELETE /api/purchases/:id: exclui uma compra pelo ID

* GET /api/purchases/:id: retorna uma compra pelo ID
